Cómo calcular la base reguladora del paro

Guia practica para estimar tu prestacion por desempleo con datos reales de nomina y cotizacion.

Inicio > Guia > Base reguladora del paro

Resumen rapido

  • La base reguladora es la referencia inicial para calcular la prestacion.
  • Se obtiene con la media de bases de cotizacion de los ultimos 180 dias cotizados.
  • No equivale al importe neto final que llega a tu cuenta.

Indice

  • Que es la base reguladora
  • Formula paso a paso
  • Ejemplo numerico completo
  • Casos borde: parcial, discontinuo, cambios de salario
  • Errores frecuentes al calcular

Que es la base reguladora del paro

Es una media de cotizaciones que sirve como base bruta para estimar la prestacion por desempleo. A partir de ese valor se aplican porcentajes legales y limites para obtener la cuantia reconocida.

Formula paso a paso

  1. Reune las bases de cotizacion de los ultimos 180 dias cotizados.
  2. Suma todas esas bases.
  3. Divide el total entre 180 para obtener la base diaria.
  4. Multiplica por 30 para una referencia mensual bruta.
  5. Aplica porcentajes por tramo y revisa topes vigentes.

Formula base: suma de bases de 180 dias / 180 = base reguladora diaria.

Si antes necesitas ordenar periodos y revisar acumulado, puedes apoyarte en la calculadora de dias cotizados.

Ejemplo numerico completo

Si la suma de bases de cotizacion de los 180 dias es 18.000 EUR:

  • Base diaria: 18.000 / 180 = 100 EUR.
  • Referencia mensual bruta: 100 x 30 = 3.000 EUR.
  • Sobre esa referencia se aplican porcentajes legales y topes.

Para una estimacion orientativa de cuantia final, puedes usar la calculadora de paro.

Casos borde que suelen generar dudas

  • Jornada parcial: las bases pueden ser menores y afectar la media final.
  • Periodos discontinuos: conviene revisar que se toman los dias cotizados correctos.
  • Cambios fuertes de salario: el promedio puede alejarse del ultimo sueldo.
  • Pluriempleo o periodos especiales: revisar bases y compatibilidades con detalle.

Errores frecuentes al calcular la base reguladora

  • Usar salario neto en lugar de bases de cotizacion.
  • Tomar meses completos en lugar de los 180 dias cotizados correctos.
  • Confundir base reguladora con importe neto final.
  • No tener en cuenta topes legales minimos y maximos.

Preguntas frecuentes

Que necesito para calcular la base reguladora?

Las bases de cotizacion de los ultimos 180 dias cotizados antes del desempleo.

Es lo mismo base reguladora que paro neto?

No. La base es un punto de partida bruto y el importe final puede variar.

Con contrato parcial cambia el calculo?

Si, porque las bases de cotizacion pueden modificar la media resultante.

Puedo estimar rapidamente cuanto cobraria?

Si, con la calculadora de paro obtienes una estimacion orientativa a partir de tus datos.

De la base reguladora al importe final

  • Cuánto se cobra de paro para aterrizar topes y tramos de pago.
  • Verificar derecho a prestación por desempleo si aun dudas si cumples requisitos.
  • Simulador de paro pendiente para seguimiento tras varios cobros.
