Como se calcula la IT de forma orientativa

  1. Base reguladora diaria: base mensual / 30.
  2. Contingencia comun: tramos habituales 1-3, 4-20 y 21+.
  3. Contingencia profesional: referencia habitual del 75% desde el dia siguiente.
  4. Total estimado: suma de tramos segun numero de dias de baja.

Preguntas frecuentes

Incluye complementos de convenio?

No. Si tu convenio mejora la IT, el cobro real puede ser superior.

Sirve para saber el neto exacto?

No. Es una referencia en bruto y no incluye todas las retenciones o ajustes.

Que pasa si la baja dura varios meses?

La herramienta estima por dias de baja informados. Si cambian bases o condiciones, conviene recalcular por periodos.

Aplica igual para autonomos?

No siempre. En autonomos pueden aplicar reglas y coberturas diferentes segun el regimen.

Puedo usar base diaria directamente?

La calculadora parte de base mensual y la convierte entre 30 para simplificar el calculo orientativo.

Es valido para reclamar?

Solo como orientacion inicial. Para reclamaciones usa nominas, convenio y asesoramiento profesional.
